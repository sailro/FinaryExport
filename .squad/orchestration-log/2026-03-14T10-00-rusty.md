## Task: Cross-Platform Support Scoping

Analyzed two Windows-only blockers to feasibility-scope cross-platform support (Windows + Linux + macOS).

**Findings:**
1. **DPAPI session encryption** — Windows-only via `System.Security.Cryptography.ProtectedData`. Cleanly abstracted behind `ISessionStore` interface. Replacement: `Microsoft.AspNetCore.DataProtection` (cross-platform, encryption at rest, small effort).
2. **CurlImpersonate TLS bypass** — NuGet package ships `linux-x64` `.so` binary (untested but present); macOS has zero binaries and no build path. `Loxifi.CurlImpersonate` readme claims Windows-only but includes Linux support.

**Recommendation:** Windows + Linux x64 feasible (~1 day total). macOS blocked by build complexity and Loxifi's upstream gaps.

**Output:** `.squad/decisions/inbox/rusty-cross-platform.md`
