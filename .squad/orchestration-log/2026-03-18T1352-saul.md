## Key Learning

MCP elicitation is a **client-advertised capability** sent during `initialize`. The SDK's `ElicitAsync` performs a runtime check but throws an opaque error if the capability is missing. Always pre-flight check `ClientCapabilities?.Elicitation?.Form` before calling `ElicitAsync` and provide clear fallback guidance.
