## Technical Details

The Finary API only supports `/portfolio/{category}/transactions` for 4 asset categories:
- Checkings
- Savings
- Investments
- Credits

The other 6 categories (Crypto, RealEstate, PreciousMetals, etc.) return HTTP 404. The new `HasTransactions()` method acts as a gatekeeper; `TransactionsSheet` now filters before querying, eliminating spurious warnings while retaining per-category try/catch as a safety net for genuine errors.
